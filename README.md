This repository contains my personal developer portfolio built using **HTML**, **CSS**, and **JavaScript**.

The website highlights my Android development journey, featured projects, technical skills, education, and contact information in a modern glassmorphism UI.

It is designed to provide recruiters and developers with a quick overview of my work and experience.
